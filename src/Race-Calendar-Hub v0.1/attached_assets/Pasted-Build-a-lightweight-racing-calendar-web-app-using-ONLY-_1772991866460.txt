RACE DATA

All race data must be stored in:

races.js

The races.js file exports a variable called:

races

Example structure:

const races = [
{
series: "F1",
race: "Australian Grand Prix",
circuit: "Albert Park",
date: "2026-03-15"
}
];

NEXT RACE LOGIC

In script.js:

1. Load the race data
2. Compare each race date with today's date
3. Find the nearest upcoming race
4. Display it in the NEXT RACE panel

Season Completed
